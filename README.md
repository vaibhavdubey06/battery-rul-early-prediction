# Battery RUL Early Prediction

This project focuses on early prediction of Remaining Useful Life (RUL) for batteries using machine learning techniques including LSTM and Transformer models.

## Project Structure

```
battery-rul-early-prediction/
├── data/                    # Dataset files
├── notebooks/               # Jupyter notebooks for analysis
├── src/                     # Source code
├── results/                 # Output plots and metrics
└── papers/                  # Reference papers
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

1. Download the dataset (see `data/dataset_link.md`)
2. Run the notebooks in order:
   - `01_data_exploration.ipynb`
   - `02_feature_engineering.ipynb`
   - `03_lstm_model.ipynb`
   - `04_transformer_model.ipynb`
