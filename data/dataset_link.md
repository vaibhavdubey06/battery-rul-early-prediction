# Dataset Links

## NASA Battery Dataset
- **Source**: NASA Prognostics Center of Excellence
- **Link**: https://www.nasa.gov/content/prognostics-center-of-excellence-data-set-repository
- **Description**: Battery aging datasets with charge/discharge cycles

## CALCE Battery Dataset
- **Source**: Center for Advanced Life Cycle Engineering
- **Link**: https://calce.umd.edu/battery-data
- **Description**: Lithium-ion battery cycling data

## Instructions
1. Download the dataset from the links above
2. Place the raw data files in this `data/` directory
3. Run the data exploration notebook to verify the data
